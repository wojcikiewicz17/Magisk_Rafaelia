## 🔄 CI/CD Integration

### GitHub Actions Workflow

Create `.github/workflows/governance-check.yml`:

```yaml
name: Governance Compliance Check

on:
  pull_request:
    branches: [ main, develop ]
  push:
    branches: [ main, develop ]

jobs:
  governance:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Python
      uses: actions/setup-python@v4
      with:
        python-version: '3.10'
    
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install hashlib blake3
    
    - name: Run governance activation
      run: |
        chmod +x ativar.py
        ./ativar.py verify -v
    
    - name: Run integrity checks
      run: ./ativar.py integrity
    
    - name: Check compliance
      run: |
        if ! ./ativar.py activate --loop 3; then
          echo "❌ Governance compliance check failed"
          exit 1
        fi
        echo "✅ Governance compliance check passed"
```

### Pre-commit Hook

Create `.git/hooks/pre-commit`:

```bash
#!/bin/bash
# Pre-commit governance check

echo "🔍 Running governance compliance check..."

if ! ./ativar.py verify; then
    echo "❌ Governance check failed. Commit blocked."
    echo "   Run './ativar.py verify -v' for details"
    exit 1
fi

echo "✅ Governance check passed"
exit 0
```

```bash
chmod +x .git/hooks/pre-commit
```
